# 맛집 분석 대시보드

이 프로젝트는 맛집 데이터를 분석하고 시각화하는 Streamlit 기반의 웹 대시보드입니다.

## 개요

이 대시보드는 네 가지 주요 분석 기능을 제공합니다:
1. 데이터 개요: 전체 데이터셋의 구조와 통계 정보 제공
2. 리뷰어 분석: 개별 리뷰어의 리뷰 패턴과 선호도 분석
3. 식당 분석: 개별 식당의 상세 정보와 리뷰 트렌드 분석
4. 카테고리 분석: 식당 카테고리별 통계와 트렌드 분석

## 환경 설정

환경 설정은 root에 있는 README.md을 참고해주세요. `pyproject.toml`에 `streamlit` 관련 패키지도 포함되어 있습니다.

## 실행 방법

```bash
poetry run streamlit run apps/main.py
```

## 기능 설명

### 데이터 개요
- 전체 데이터셋 구조 설명
- 테이블별 스키마 정보
- 데이터 규모 및 통계
- 리뷰 분포 분석

### 리뷰어 분석
- 리뷰어 ID 기반 검색
- 리뷰 통계 분석
- 선호 카테고리 분석
- 시간별 리뷰 패턴 분석
- 메뉴 선호도 분석

### 식당 분석
- 식당명 또는 카테고리 기반 검색
- 기본 정보 및 메뉴 정보 표시
- 리뷰 트렌드 분석
- 위치 정보 표시

### 카테고리 분석
- 계층적 카테고리 분석
- 카테고리별 통계
- 트렌드 분석
- 결측치 분석
